今天开始做整个项目：首先，需要对项目进行分析
	1、在整个项目需要创建的表有哪些，以及其中的字段
		1> Users	
				字段： 
					 id 					--> 用户编号，主键（这里采用序列自动增长） 
				     name					--> 登录名字
				     password				--> 登录密码
		2> Admins
				字段：
					 id						--> 管理员编号，主键（这里采用序列自动增长）
				     name					--> 登录名字
				     password				--> 登录密码
		3> News
				字段：
					 id						--> 新闻编号，主键
				     title					--> 新闻标题
				     author					--> 新闻作者
				     title					--> 新闻内容
		4> Goods
				字段：
					 id						--> 商品编号
					 name					--> 商品名称
					 category				--> 商品类别（这里和商品类别表，一对多关系）
					 desc					--> 商品描述
					 
		5> GoodsCategory
				字段：   
					 id                     --> 类别编号
					 name					--> 类别名称
